freelancer-corps-api Documentation

The Freelancer Corps API is a Demon Slayer–themed freelancing job portal backend. It powers a full‑stack system where Slayers (freelancers) take on missions posted by an Admin, battling demons, gaining experience, and ranking up based on completed work.

This API manages users, demons, missions, reports, authentication, and rankings. Slayers can browse missions, accept those within their rank, and submit mission reports. Admins create missions, register demons, and oversee corps operations. All authentication uses JWT and passwords are secured using bcrypt.

Missions are themed after Demon Slayer lore. Each mission targets a specific demon, and every demon has its own rank and power level. Missions also grant EXP rewards, allowing Slayers to advance their professional rank, proving their reliability based on past performance.

User Roles:

  • Admin – Can create, update, and manage missions and demons. Has elevated permissions.
  • Slayer – A freelancer who accepts missions, submits reports, and grows through ranking and experience.

Slayer Ranks:

  • Mizunoto – Novice
  • Tsuchinoe – Mid Rank
  • Hashira – Elite

Slayers can only take missions that match their difficulty level and rank. Demon power levels help Slayers determine whether they can handle specific missions. After completing or failing a mission, Slayers must submit a mission report, enabling transparent tracking of their performance.

This API is fully structured with RESTful design principles and serves as the backend for the future Freelancer Corps full‑stack application.

Base URL

http://localhost:10000/

Authentication

POST /auth/login

Logs in a user and returns a JWT token.

Request Body

{
  "email": "string",
  "password": "string"
}

Response

{
  "message": "Login successful",
  "token": "",
  "user": { "_id": "string", "email": "string", "role": "admin | slayer" }
}

GET /auth/admin

Admin protected route.

Response

{ "message": "Welcome Admin", "user": { ... } }

GET /auth/user

Slayer protected route.

Response

{ "message": "Welcome User", "user": { ... } }

Demons

POST /demons

Create a new demon (Admin only).

Request Body

{
  "name": "Akaza",
  "rank": "Upper Moon 3",
  "powerLevel": 9500
}

GET /demons

Fetch all demons.

GET /demons/:demonID

Fetch demon by ID.

PATCH /demons/:demonID

Update demon (Admin only).

DELETE /demons/:demonID

Delete demon (Admin only).

Missions

POST /missions

Create mission (Admin only).

{
  "title": "Eliminate the Swamp Demon",
  "difficulty": "Mid",
  "expReward": 300,
  "assignedTo": "slayerUserID",
  "demonID": "demonID"
}

GET /missions

Fetch all missions.

GET /missions/:missionID

Fetch mission by ID.

PATCH /missions/:missionID

Update mission (Slayer only).

DELETE /missions/:missionID

Delete mission (Admin only).

Reports

POST /reports

Create report (Admin only).

GET /reports

Fetch all reports.

GET /reports/:reportID

Fetch report by ID.

PATCH /reports/:reportID

Update report (Slayer only).

DELETE /reports/:reportID

Delete report (Admin only).

Users

POST /users/register

Create a new user.

{
  "email": "string",
  "password": "string",
  "role": "admin | slayer"
}

GET /users

Fetch all users (Admin only).

GET /users/profile

Fetch all profiles.

GET /users/:userID

Fetch user by ID (Admin only).

GET /users/:userID/profile

Fetch single user profile.

PATCH /users/:userID/profile

Update profile (Slayer only).

DELETE /users/:userID

Delete user (Admin only).

GET /users/:userID/missions

Get user missions.
